# 配置表说明（关卡波次 / 敌人 / Boss / 难度）

> 本文件夹内的 4 个 `.csv` 即"Excel 配置表"，**双击可直接用 Excel / WPS / 表格软件打开**（已带 UTF-8 BOM，中文不乱码）。如需 `.xlsx`，在 Excel 中"另存为 → Excel 工作簿(.xlsx)"即可，也可把 4 个表导入同一工作簿的 4 个 Sheet。

## 文件清单

| 文件 | 作用 | 对应 Sheet |
| --- | --- | --- |
| `关卡波次配置表.csv` | 3 关全部刷怪波次（导入/推进/精英/Boss 段） | 波次 |
| `敌人属性表.csv` | 6 种杂兵 + 3 种精英的属性 | 敌人 |
| `Boss配置表.csv` | 3 个 Boss 的阶段与机制 | Boss |
| `难度系数表.csv` | 简单/普通/困难全参数 | 难度 |

## 字段说明

### 关卡波次配置表

| 列 | 含义 |
| --- | --- |
| 关卡 | 所属关卡 |
| 波次ID | 唯一标识（L1-W01 等），Boss 段为 L1-B |
| 触发时间s | 关卡计时到达该秒数时生成本波 |
| 段落 | 导入段 / 推进段 / 精英 / Boss前补给 / Boss段 |
| 敌人ID | 关联《敌人属性表》或《Boss配置表》 |
| 数量 | 本波生成数量 |
| 生成位置 | 顶部均匀/两侧/随机/全屏/屏幕中上 等 |
| 生成间隔s | 同波内逐个生成的间隔 |
| 移动模式 / 弹幕模式 | 行为标识，程序按枚举实现 |
| 掉落道具 / 掉落率 | 击杀掉落配置（受难度系数影响） |
| 关联科普ID | 触发或解锁的科普卡编号 |
| 备注 | 设计意图 |

### 敌人属性表

`敌人ID / 名称 / 关卡 / 基础血量 / 移动速度 / 分值 / 移动模式 / 弹幕模式 / 开火间隔 / 击中半径 / 关联科普ID / 病理隐喻`
> 实际血量 = 基础血量 × 难度血量系数。

### Boss配置表

`BossID / 名称 / 关卡 / 总血量 / 阶段 / 阶段血量区间 / 弹幕模式 / 特殊机制 / 弱点部位 / 关联科普ID`
> 一个 Boss 占多行（每行一个阶段）。

### 难度系数表

每个参数三档取值 + 说明；由程序的 `DifficultyService` 全局注入到各战斗模块。

## 与 Unity 的对接

1. 策划在 CSV 中填表 / 调数值。
2. 程序提供编辑器导入工具（见《Unity工程架构与代码指导》第七章），菜单一键把 CSV 解析为 `LevelTimeline` / `EnemyData` / `DifficultyData` 等 ScriptableObject。
3. "敌人ID"字符串在导入时映射到对应 `EnemyData` 资产。
4. 调整内容只改 CSV + 重新导入，**不改代码**。

## 注意事项

- CSV 单元格内**不要使用英文逗号**（会被当作分隔符）；需要时用中文逗号或在 Excel 中保存为带引号格式。
- 时间列在同一关卡内应**递增**。
- 数值平衡需多次手测，尤其困难档需保证弹幕"有解"。
- 科普ID 对应策划案第七章知识点清单（K01–K10）。
