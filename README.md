## 📖 一句话简介

> 你是注入患者体内的纳米战机驾驶员，在血管、器官与神经的黑暗深处，击退因糖尿病而失控、异变、增殖的"高糖怪物"，拯救一具濒临崩溃的身体。

## 🎮 游戏概述

| 维度 | 说明 |
|------|------|
| **类型** | 纵版卷轴飞行射击（STG / 弹幕射击，"雷霆战机"-like） |
| **引擎** | Unity 2022 LTS（URP 2D Renderer） |
| **平台** | PC（Windows）首发，预留移动端触控适配 |
| **题材** | 糖尿病科普 · 人体微观世界 · 科幻 / 黑暗 / 恐怖 |
| **目标受众** | 12 岁以上科普向玩家、健康教育课堂、医院/社区健康宣传 |
| **目标周期** | 2 周（10 个工作日）原型可玩版 |
| **团队规模** | 1-3 人 |

### 核心卖点

- 🧬 **题材即玩法**：每个敌人、每张地图、每次死亡都在讲述真实的糖尿病病理故事
- 🩸 **生物恐怖美学**：血管、视网膜、胰岛——三大关卡递进，从黏腻血腥到衰败绝望
- 🎓 **寓教于乐**：通关解锁病理档案卡，死亡读条科普提示，医学院级别的知识植入
- ⚡ **数据驱动**：ScriptableObject 配置一切，策划改 CSV 即可调数值，不改代码

## 🎨 设计哲学

### 三大原则
1. **数据驱动**：敌人、弹幕、关卡波次、难度、科普全部用 ScriptableObject 配置，CSV 导入，改内容不改代码
2. **事件解耦**：玩法层通过事件总线通知 UI/音频，避免互相直接引用
3. **对象池优先**：子弹、敌人、特效一律池化，杜绝运行时频繁 Instantiate/Destroy

### 技术栈
| 项 | 选择 | 说明 |
|------|------|------|
| Unity | 2022 LTS | 稳定，长期支持 |
| 渲染管线 | URP（2D Renderer） | 后处理（泛光/暗角/噪点）营造生物恐怖氛围 |
| 输入 | Input System（新版） | 统一键盘/手柄/触屏 |
| UI | UGUI（Canvas） | 上手快，够用 |
| 配置 | ScriptableObject + CSV | 策划可在 Excel 填表后一键导入 |
| 存档 | PlayerPrefs / JSON | 进度、解锁、排行榜 |
| 版本管理 | Git + Git LFS | LFS 管理美术资产 |

### 三层关卡递进
```
关一《血潮》血管  ──→  关二《盲视》眼底  ──→  关三《核心》胰岛
   高血糖黏稠            视网膜病变              胰岛功能衰竭
   暗红血浆             墨黑血网                病态坏死
```
