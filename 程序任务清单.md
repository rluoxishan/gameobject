# 《糖战·胰岛防线》程序任务清单

> **配套**：游戏策划案 + Unity 工程架构与代码指导文件 + 关卡波次配置表
> **引擎**：Unity 2022 LTS + URP(2D Renderer) + Input System
> **方法**：数据驱动（ScriptableObject 配置敌人/关卡/弹幕/科普），全量对象池。
> **团队规模假设**：1-3 人，10 个工作日。

---

## 一、任务优先级与状态字段说明

- **优先级**：P0=核心可玩必需 / P1=内容完整必需 / P2=包装可裁剪。
- **状态**：未开始 / 进行中 / 已完成 / 阻塞。
- **预估**：理想人时（h）。
- **依赖**：前置任务编号。

## 十四、关键技术风险与对策

| 风险 | 对策 |
| --- | --- |
| 弹幕性能（大量子弹） | 全量对象池 + 简化判定 + 限制同屏上限 |
| 弹幕平衡难调 | 数据驱动 SO + DDA 兜底 + 多次手测 |
| CSV/SO 数据流打通慢 | 优先做 T-32 导入工具，后续配置零代码 |
| Boss 复杂度高 | 统一状态机框架(T-40)，3 个 Boss 复用 |
| 工期紧 | P2 全部可裁剪；关二/三复用素材与逻辑 |
